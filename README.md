# FSM Finance Dashboard

Simple web interface for uploading and analyzing financial statements across FSM portfolio entities.

## Supported Organizations
- HSO / Express Mart
- 303 Software  
- NORTH Dispensaries
- Lucky Services
- Minted Ventures
- Tom's Vending

## Features
- ✅ Clean upload interface
- ✅ File validation (Excel, CSV, PDF)
- 🚧 Smart financial parsing (in development)
- 🚧 Multi-entity dashboard (planned)
